Application Usage

Activity Predictor Usage

Peptide Sequences must be entered in FASTA format only. The output will display all the predicted activity labels of the peptide.

Dataset Usage

Our dataset contains a collection of more than 3000 peptides comprising both AMPs and Non-AMPS. It has been curated from multiple sources like Uniprot, APD3, AVPpred, among others.

Welcome to our peptide activity predictor and peptide database home

M-CAMP aims to classify peptides into various labels so as to understand the type of activity the peptide has and to help pharmacists and genomic scientists to model drugs which can work towards eliminating diseases. It also helps drive reproducible research in drug discovery.

Classification will be based on sequence data provided of each peptide as input.The final output will be an open source software or a web application which classifies peptides as antiviral, antimicrobial, antifungal or a combination of the three.

M-CAMP makes the work of drug discovery easier. It carries an Open Source license and is aimed at helping scientists conduct research free of cost.

Comprehensive documentation of the database can be obtained from the Help page.

M-CAMP Features

  • Access AMP and Non-AMP sequences
  • View source organisms, sequence length, and other features of each peptide
  • Classify peptide sequences based on Activity
  • Predicts single as well as multiple activity labels of peptide sequences.
